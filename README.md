# home-grandlisboa-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容多样，不针对任何单一域名或具体网站，仅作为个人笔记与资料整理的一部分。

## 项目简介

`home-grandlisboa-notes` 是一个静态页面集合仓库，主要存放通过 HTML 编写的各类记录、参考或展示页面。每个页面相对独立，不依赖外部数据库或后端服务，可直接在浏览器中打开浏览。

该仓库适合存放：
- 学习笔记与知识整理
- 工具或资源索引
- 实验性或临时性页面
- 其他可独立呈现的 HTML 内容

## 目录说明

```
home-grandlisboa-notes/
├── index.html          # 仓库首页或导航页（可选）
├── page-1.html         # 示例页面
├── page-2.html         # 示例页面
└── ...                 # 其他 HTML 文件
```

- 根目录下存放所有 HTML 页面文件，文件名通常反映页面主题或用途。
- 可根据需要创建子目录进行分类，但建议保持结构简洁。
- 每个页面应尽量自包含，避免引用外部资源（除非必要）。

## 页面归档说明

- 所有页面均为静态 HTML，可使用任何现代浏览器直接打开。
- 页面内容可能包含文字、图片、链接、表格、样式等常见元素。
- 部分页面可能引用本地资源（如图片、CSS、JS 文件），请确保文件路径正确。
- 页面之间无强制依赖关系，可单独使用。

## 维护说明

- 本仓库不定期更新，新增或修改页面会直接提交。
- 欢迎 fork 或 clone 到本地使用，但请注意内容可能随时调整。
- 如有问题或建议，请通过仓库 Issues 进行反馈。
- 不接受未经许可的广告或营销内容提交。
